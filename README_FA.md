## معرفی

**IronPanel v17** یک سیستم کامل برای مدیریت و فروش سرویس‌های VPN است. این پروژه برای مدیران سرور، فروشندگان VPN، تیم‌های فنی و ارائه‌دهندگان سرویس طراحی شده و هدف آن این است که مدیریت چند پروتکل، چند نود، کاربران، فروش، مانیتورینگ و سابسکریپشن را در یک پنل واحد و منظم انجام دهد.

در نسخه 17 تمرکز اصلی روی چند بخش مهم بوده است: سیستم نود حرفه‌ای، پایدارسازی Xray/V2Ray، سابسکریپشن پیشرفته، Outbound Routing، ربات فروش، مانیتورینگ زنده، بکاپ/ریستور و ابزارهای مدیریتی بهتر.

---

## امکانات اصلی

### مدیریت چند پروتکل

IronPanel از چند پروتکل VPN در یک پنل واحد پشتیبانی می‌کند:

- OpenVPN
- WireGuard
- Cisco AnyConnect / Ocserv
- L2TP/IPsec
- Xray / V2Ray

مدیر می‌تواند برای هر کاربر محدودیت حجم، تاریخ انقضا، وضعیت فعال یا غیرفعال، دسترسی به پروتکل‌ها و فایل‌های اتصال را مدیریت کند.

---

## OpenVPN

بخش OpenVPN برای تحویل کانفیگ پایدار و مناسب کاربران طراحی شده است.

ویژگی‌ها:

- تولید فایل کانفیگ اختصاصی برای هر کاربر
- نام فایل مطابق نام کاربر
- پشتیبانی از TCP و UDP طبق تنظیمات مدیر
- ساخت کانفیگ certificate-only
- عدم نیاز به وارد کردن username/password در کلاینت
- ثبت ترافیک آپلود و دانلود
- تشخیص کاربران آنلاین
- بررسی وضعیت سرویس OpenVPN
- ابزار Repair برای رفع خطاهای رایج
- هماهنگی با سیستم حجم و انقضا

---

## WireGuard

ویژگی‌های WireGuard:

- ساخت کلید اختصاصی برای هر کاربر
- تولید فایل کانفیگ WireGuard
- تولید QR Code
- نمایش وضعیت اتصال و آخرین Handshake
- محاسبه مصرف کاربر
- ریست ترافیک
- هماهنگی با سابسکریپشن و ربات فروش

---

## Cisco AnyConnect / Ocserv

ویژگی‌ها:

- مدیریت کاربران Ocserv
- نمایش وضعیت سرویس
- پشتیبانی از TCP/UDP بر اساس تنظیمات سرویس
- هماهنگی با محدودیت حجم و تاریخ انقضا
- امکان نمایش اطلاعات اتصال در سابسکریپشن
- Health Check و Repair

---

## L2TP/IPsec

ویژگی‌ها:

- پشتیبانی از L2TP/IPsec
- مدیریت کاربران
- ثبت وضعیت سرویس
- هماهنگی با سیستم حجم و انقضا
- مناسب برای دستگاه‌هایی که از پروتکل‌های کلاسیک پشتیبانی می‌کنند

## Xray / V2Ray Core

از نسخه 16 به بعد، هسته Xray به IronPanel اضافه شد و در نسخه 17 پایدارتر و کامل‌تر شده است.

### پروفایل‌های پشتیبانی‌شده

- VLESS + Reality + Vision
- VLESS + WebSocket + TLS
- VLESS + TCP بدون TLS
- VLESS + WebSocket بدون TLS
- VLESS + gRPC بدون TLS
- VMess + WebSocket
- VMess + TCP بدون TLS
- Trojan + TLS
- Trojan + TCP بدون TLS
- Shadowsocks

### قابلیت‌های Xray / V2Ray

- انتخاب فقط یک نوع کانفیگ فعال توسط مدیر
- تحویل فقط همان کانفیگ انتخاب‌شده به کاربر
- ساخت UUID اختصاصی برای هر کاربر
- تولید Reality Key و Short ID
- تولید QR Code
- خروجی لینک خام Xray
- تولید فایل xray.txt
- اعتبارسنجی کانفیگ قبل از اعمال
- بررسی سلامت سرویس Xray
- ابزار Repair اختصاصی Xray
- پشتیبانی از حالت‌های TLS، Reality و بدون TLS
- ثبت مصرف کاربران
- هماهنگی با ربات فروش
- هماهنگی با سابسکریپشن‌های مختلف

## Outbound Routing

بخش Outbound Routing برای زمانی طراحی شده که مدیر بخواهد ترافیک یک یا چند پروتکل را از یک کانفیگ خروجی دیگر عبور دهد.

امکانات:

- وارد کردن کانفیگ OpenVPN Client به عنوان خروجی
- وارد کردن کانفیگ V2Ray / Xray به عنوان خروجی
- پشتیبانی از لینک‌های vless، vmess، trojan و ss
- تست اتصال قبل از اعمال تنظیمات
- انتخاب پروتکل‌هایی که باید از Outbound عبور کنند
- پشتیبانی از OpenVPN، WireGuard، Ocserv، L2TP و Xray
- چند Outbound Profile
- Failover
- Kill Switch
- انتخاب Route Mode
- بررسی IP خروجی و وضعیت اتصال
