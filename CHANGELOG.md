# Changelog

## v16.7

- Added Outbound Routing Manager for all license types.
- Admin can paste OpenVPN client config or V2Ray/Xray URI as upstream outbound.
- Added outbound connection test before applying runtime routing.
- Added protocol selection for outbound traffic: OpenVPN, WireGuard, Ocserv, L2TP/IPsec and Xray/V2Ray.
- Added managed OpenVPN outbound client service.
- Added policy routing helper for selected VPN user subnets.
- Added Xray outbound injection for upstream V2Ray/Xray URI.
- Added `/outbound` panel page and sidebar menu entry.
- Updated license feature flags so outbound is available on beginer, plus, pro, admin and trial.
- Updated LicensePanel feature response with outbound support for all license types.
