# v19.0.0 - Famous DNS Presets

این نسخه DNSهای معروف را به‌صورت خودکار به بخش DNS Manager و WireGuard DNS اضافه می‌کند.

## DNSهای آماده

- Cloudflare: `1.1.1.1, 1.0.0.1`
- Google: `8.8.8.8, 8.8.4.4`
- Quad9: `9.9.9.9, 149.112.112.112`
- OpenDNS: `208.67.222.222, 208.67.220.220`
- AdGuard DNS: `94.140.14.14, 94.140.15.15`
- DNS.SB: `185.222.222.222, 45.11.45.11`
- Shecan: `178.22.122.100, 185.51.200.2`

## رفتار آپگرید

در `init-db` و `upgrade-db` پروفایل‌های DNS معروف اضافه/به‌روزرسانی می‌شوند، اما DNSهای دلخواه ادمین حذف نمی‌شوند.
