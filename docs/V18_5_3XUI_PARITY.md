# IronPanel v18.5.0 - 3x-ui Parity UX Pack

این نسخه برای کم‌کردن فاصله با پنل‌های تخصصی Xray مثل 3x-ui ساخته شده و روی سادگی ساخت کانفیگ و امکانات حرفه‌ای‌تر تمرکز دارد.

## تغییرات اصلی

- Quick Create User برای ساخت سریع کاربر با preset آماده.
- Xray Pro Builder با چند Inbound همزمان و خروجی چند لینک در فایل `xray.txt`.
- Reality / TLS / WS / gRPC / Trojan / Shadowsocks در حالت builder.
- IP Limit per User برای جلوگیری از اشتراک‌گذاری اکانت.
- GeoFile Manager برای آپدیت `geoip.dat` و `geosite.dat`.
- Subscription Theme Manager برای برند، متن راهنما، رنگ و نمایش/عدم نمایش raw config.
- Admin Telegram Bot settings و گزارش مدیریتی تست.
- User search در لیست کاربران.

## نکته سازگاری

برای جلوگیری از خراب‌شدن نصب‌های قبلی، تنظیمات جدید تا جای ممکن در `AppSetting` ذخیره شده‌اند و نیازمند migration سنگین دیتابیس نیستند.
