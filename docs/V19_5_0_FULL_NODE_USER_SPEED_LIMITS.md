# v19.5.0 - Full Node Manager + Per-User Speed Limits

- Node Gateway تکمیل شد: انتخاب نود ثابت، Auto Balance، بهترین پینگ، کمترین کاربر، امتیاز ترکیبی و Failover.
- Node Agent ساده‌تر شد و با یک دستور نصب می‌شود.
- Heartbeat نود حالا protocol health، online users و metrics ارسال می‌کند.
- Job queue برای sync کاربر، health check و repair/install core روی نود اضافه شد.
- کاربران می‌توانند روی Local/Auto/Fixed Node تنظیم شوند.
- محدودیت سرعت از حالت کلی پروتکل به حالت هر کاربر روی هر پروتکل تغییر کرد.
- README طبق ساختار فارسی پروژه به‌روزرسانی شد.
