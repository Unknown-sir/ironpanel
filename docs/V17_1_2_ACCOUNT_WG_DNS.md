# IronPanel v17.1.2 - Account Settings + WireGuard DNS

- Added `/account` page for main admins and resellers to change their own username/password.
- Added main-admin reseller delete action; owned users are detached by default instead of deleted.
- Added configurable WireGuard DNS setting. Generated client configs now use `wireguard_dns` instead of a hardcoded DNS.
- Updated README and README_FA with the new account, reseller delete and WireGuard DNS notes.
