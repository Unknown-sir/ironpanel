# v17.1.1 - Subscription Popup UI

در این نسخه صفحه Subscription خلوت‌تر شد. به‌جای نمایش همه کانفیگ‌ها در صفحه، فقط کارت/آیکون هر پروتکل نمایش داده می‌شود و با کلیک روی هر پروتکل، پاپ‌آپ همان کانفیگ باز می‌شود.

## تغییرات

- نمایش آیکون/کارت جدا برای هر پروتکل.
- باز شدن کانفیگ هر پروتکل داخل Pop-up.
- دکمه Copy داخل پاپ‌آپ.
- دکمه Download File داخل پاپ‌آپ.
- نمایش QR داخل پاپ‌آپ برای WireGuard، Xray و Hysteria2 در صورت وجود.
- کاهش شلوغی صفحه Subscription مخصوصاً روی موبایل.
- بهبود Responsive صفحه ساب.
