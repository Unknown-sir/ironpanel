# IronPanel v18.3

This release fixes the v18 dashboard metric widgets and aligns the LicensePanel UI with the IronPanel v18 VPN-UI theme.

## Fixed
- CPU usage display
- RAM usage display
- Swap usage display
- Disk usage display
- License remaining days widget
- Safe dashboard rendering when the metrics API is temporarily unavailable

## LicensePanel
- Updated visual style to the IronPanel v18 dark/teal dashboard design
- Kept existing license, sales bot, plan, order and activation workflows unchanged
