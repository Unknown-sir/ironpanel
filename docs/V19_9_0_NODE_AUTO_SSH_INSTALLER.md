# IronPanel 19.9.0 — Node Auto SSH Installer

این نسخه نصب نود را از داخل پنل خودکار می‌کند.

## امکانات

- نصب خودکار نود از طریق SSH از داخل پنل اصلی.
- پشتیبانی از ورود با Password یا SSH Private Key.
- ذخیره اطلاعات ورود به‌صورت رمزنگاری‌شده، نه Hash و نه متن خام.
- فایل کلید رمزنگاری محلی: `/etc/ironpanel/node_credential.key` با permission پیشنهادی `600`.
- اجرای نصب، نصب/Repair هسته‌ها، Sync کانفیگ‌ها، Sync کاربران و Health Check بعد از نصب.
- نمایش لاگ نصب در صفحه نود.
- قابلیت پاک‌کردن credential ذخیره‌شده از پنل.
- محدودیت دسترسی فقط برای لایسنس‌های Pro و Admin.

## چرا Hash نه؟

برای اتصال SSH، پنل باید بتواند پسورد یا کلید خصوصی را دوباره به مقدار اصلی تبدیل کند. Hash فقط برای بررسی رمز عبور کاربرد دارد و قابل بازیابی نیست. بنابراین این نسخه از encryption استفاده می‌کند.

## پکیج‌های جدید

- `paramiko` برای SSH
- `cryptography` برای رمزنگاری credentialها
