# v19.4.1 — Responsive Users & Configs UI

- User list no longer renders long protocol strings inside a wide table.
- Each user is displayed as a responsive card with essential usage/status info.
- Protocols are hidden behind a compact `!` button and shown on click.
- User actions wrap into a compact responsive grid without horizontal scrolling.
- User config pages are cleaner on mobile and long config bodies are safely wrapped.
