# IronPanel v17.0.0 — SSH Protocol

این نسخه پروتکل `ssh` را مثل ShahanPanel به IronPanel اضافه می‌کند.

- پروتکل جدید: `ssh`
- پورت پیش‌فرض: `422/tcp`
- اسکریپت repair: `/opt/ironpanel/scripts/repair_ssh.sh --sync`
- خروجی کاربر: `ssh.txt`
- اکانت‌های SSH در گروه `ironpanel-ssh` ساخته می‌شوند و برای تونل محدود شده‌اند.
